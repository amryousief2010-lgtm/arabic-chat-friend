# بناء نظام مزرعة الأمهات + معمل التفريخ

نظام تشغيلي متكامل يحاكي ملف الإكسل المرفق، يربط إنتاج البيض في المزرعة → النقل → التفريخ → خروج الكتاكيت، مع لوحة تحكم ومقارنات جودة ومطابقة.

## 1. قاعدة البيانات (جداول جديدة)

### مزرعة الأمهات
- **`farm_families`** (الأسر): رقم الأسرة، الملعب، الحالة، عدد الإناث، عدد الذكور، تاريخ بدء التشغيل، ملاحظات
- **`farm_egg_production`** (إنتاج البيض اليومي): التاريخ، family_id، عدد البيض، ملاحظات
- **`farm_transfers`** (نقل البيض للمعمل): التاريخ، الكمية المنقولة، الهالك، الأسرة (اختياري)
- **`farm_feed_log`** (سجل العلف): التاريخ، النوع، الكمية المصروفة، الملاحظات
- **`farm_medications`** (الأدوية واللقاحات): التاريخ، اسم الدواء، الجرعة، الأسرة، ملاحظات

### معمل التفريخ
- **`hatch_customers`** (عملاء المعمل): الاسم، النوع (داخلي/خارجي)، سعر التحضين، سعر غير المخصب، سعر الهاتشر
- **`hatch_batches`** (دفعات المعمل): رقم الدفعة، التاريخ، customer_id، وارد، صافي، تاريخ الدخول، الكشف الأول، الكشف الثاني، الخروج، النافق، الكتاكيت، الحالة، الماكينة
- **`hatch_daily_ops`** (التشغيل اليومي): التاريخ، الحالة، الطاقة، الملاحظات
- **`hatch_maintenance`** (صيانة المعمل): التاريخ، النوع (دورية/طارئة)، الماكينة، الإجراء، التكلفة
- **`chick_movements`** (حركة الكتاكيت): التاريخ، المصدر، وارد، منصرف، نافق، البيع، السعر

### إعدادات
- **`farm_hatch_settings`** (الإعدادات العامة): سنة، شهر معروض، مدة كشف أول/ثاني/خروج/هاتشر، أسعار افتراضية

كل الجداول مع RLS:
- مدير عام / تنفيذي / مدير مزرعة / مدير إنتاج / مدير جودة → كل الصلاحيات
- باقي الأدوار → قراءة فقط

## 2. صفحة مزرعة الأمهات (`/modules/farm`)

تبويبات (Tabs):
1. **لوحة المزرعة**: إجمالي الأسر، عدد الإناث/الذكور، بيض الشهر، بيض YTD، بيض/أنثى، آخر تحديث
2. **الأسر** (Farm_Master): جدول CRUD للأسر مع فلترة بالحالة
3. **إنتاج البيض اليومي** (Farm_Eggs): إدخال يومي + جدول + فلترة بالشهر + رسم بياني
4. **نقل البيض للمعمل** (Farm_Transfers): إدخال + جدول + إجمالي المنقول
5. **سجل العلف**
6. **الأدوية واللقاحات**

## 3. صفحة معمل التفريخ (`/modules/hatchery`)

تبويبات:
1. **لوحة المعمل**: دفعات نشطة، خصوبة العاصمة، خصوبة العملاء، كتاكيت الشهر، نافق هاتشر
2. **العملاء** (Hatch_Customers)
3. **الدفعات** (Hatch_Batches): إدخال دفعة كاملة + تحديث المراحل (كشف أول/ثاني/خروج) + حسابات الخصوبة والفقس تلقائياً
4. **التشغيل اليومي**
5. **الصيانة**
6. **حركة الكتاكيت**

## 4. صفحة لوحة التحكم الموحدة (`/modules/farm-hatchery-dashboard`)

تجمع المؤشرات من المزرعة والمعمل:
- بيض المزرعة الشهر/YTD
- منقول للمعمل الشهر/YTD
- خصوبة العاصمة (داخلي) vs خصوبة الآخرين
- التحول للكتكوت (لكل مجموعة)
- فرق النقل/الوارد (مطابقة)
- مقارنة الجودة (Recharts BarChart)
- آخر تواريخ التحديث

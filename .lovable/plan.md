
# خطة استيراد وعرض بيانات تحليل مبيعات 2025

## ملخص البيانات في الملف
الملف يحتوي على تحليل شامل لـ 7,995 طلب بإجمالي مبيعات 13,440,427 ج.م لعام 2025، ويتضمن:
- ملخص المبيعات الشهرية (12 شهر)
- المبيعات حسب المحافظة (~35 محافظة)
- مصادر العملاء (واتساب، فيسبوك، تيك توك، إلخ)
- شركات الشحن (زودكس، مندوب خاص، العاصمة، إلخ)
- أداء الموديراتور (أية، هبة، رانيا، سارة، إلخ)
- تحليل المنتجات (~35 منتج مع الكميات)
- بيانات الطلبات التفصيلية (7,995 سجل)

## الوضع الحالي في قاعدة البيانات
- 1,076 عميل موجود (من استيراد سابق لـ 394 طلب فقط)
- 394 طلب فقط من أصل 7,995

## المرحلة 1: استيراد بيانات العملاء والطلبات المتبقية

### 1.1 إنشاء Edge Function محسّنة للاستيراد الضخم
- تعديل `supabase/functions/import-sales/index.ts` لمعالجة البيانات الجديدة من الملف
- الملف يحتوي على أعمدة إضافية مهمة: المحافظة، مصدر العميل، شركة الشحن، الموديراتور، تفاصيل المنتجات
- معالجة الـ 7,995 طلب على دفعات (batches) لتجنب timeout
- تخطي الطلبات المستوردة سابقا (394 طلب)
- توحيد العملاء المكررين بناء على رقم الهاتف

### 1.2 تحديث جدول العملاء بالبيانات الجديدة
- إضافة/تحديث حقل `city` (المحافظة) لكل عميل
- تحديث `total_orders` و `total_spent` بالقيم الصحيحة
- إضافة العنوان التفصيلي من البيانات

### 1.3 إضافة أعمدة جديدة لجدول الطلبات (migration)
- إضافة عمود `source` (مصدر العميل: فيسبوك، واتساب، إلخ)
- إضافة عمود `shipping_company` (شركة الشحن)
- إضافة عمود `moderator` (اسم الموديراتور)

## التفاصيل التقنية

### Migration SQL:
```sql
ALTER TABLE orders ADD COLUMN IF NOT EXISTS source text;
ALTER TABLE orders ADD COLUMN IF NOT EXISTS shipping_company text;
ALTER TABLE orders ADD COLUMN IF NOT EXISTS moderator text;
```

### ملفات سيتم تعديلها:
1. `supabase/functions/import-sales/index.ts` - تحسين للاستيراد الضخم (7,995 طلب)
2. `src/data/mockData.ts` - استبدال بالبيانات الحقيقية من التحليل
3. `src/pages/Index.tsx` - تحديث لوحة التحكم بجلب البيانات من قاعدة البيانات + بطاقات جديدة
4. `src/pages/Reports.tsx` - إعادة بناء شاملة بـ 6 رسوم بيانية جديدة تعكس بيانات الملف
5. `src/pages/ImportSalesData.tsx` - تحديث لدعم الاستيراد الضخم

### ملفات جديدة:
1. `src/hooks/useSalesAnalytics.tsx` - Hook لجلب وتحليل بيانات المبيعات من قاعدة البيانات
